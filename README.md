# jsflPanel
plugin  for flash IDE, can run jsfl command  
![](http://gtms04.alicdn.com/tps/i4/TB1QJTaIFXXXXX.XXXX.wjfWpXX-417-349.png)
## usage
1. download and install [jsflPanel.zxp](https://github.com/06wj/jsflPanel/blob/master/jsflPanel.zxp?raw=true).
2. click Window>Extensions>jsflPanel in the Flash Pro menu to open the jsflPanel.
3. enter jsfl commands in the textarea and click the run button to run command.
